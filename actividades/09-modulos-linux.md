# Actividad 09 — Módulos en Linux

**Fechas:** 15-22 abril 2026

## Descripción
Realizar ejercicios prácticos con módulos del kernel en un entorno Linux nativo o Live.

## Requisitos
- Instalar `build-essential` y `linux-headers`
- Crear un módulo simple `hello.c` y compilarlo
- Usar `insmod`, `lsmod`, `modinfo`, `rmmod` y `dmesg`

## Entregables
- Capturas de pantalla de cada paso
- Código del módulo (`hello.c`) y Makefile
