# Actividad 01 — Comandos Básicos del MS-DOS

**Fechas:** 13-14 enero 2026

## Descripción
Estudiar y documentar comandos básicos de la línea de comandos MS-DOS/Windows y sus variantes.

## Objetivos
- Explicar uso y opciones comunes de comandos de red, archivos y utilidades.
- Incluir ejemplos prácticos y casos de uso.
- Entregar una libreta Jupyter (.ipynb) con ejemplos ejecutables.

## Contenido (resumen)

### Red
- `ipconfig` — Muestra/renueva configuración IP
- `netstat` — Muestra conexiones de red y puertos
- `ping` — Prueba de conectividad ICMP
- `route` — Consulta y modifica la tabla de rutas

### Sistema de archivos
- `dir` — Lista archivos y carpetas
- `mkdir` — Crear directorio
- `cd` — Cambiar directorio
- `rmdir` — Eliminar directorio
- `del` — Eliminar archivos
- `type` — Mostrar contenido de archivo

### Utilidades
- `notepad` — Editor de texto
- `date` — Mostrar/establecer fecha
- `find` / `findstr` — Buscar texto en archivos

### Tuberías y redirección
- `|` tubería: pasar salida a otro comando
- `>` redirección de salida a archivo
- `<` redirección de entrada desde archivo
